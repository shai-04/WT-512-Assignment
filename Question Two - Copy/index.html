Chances are that you've already entered a few scores. Here's an explanation as to what the buttons below do.

Display Scores: Does exactly what you think it does, it displays the scores you've entered.

Display Average Score: Calculates the average of the scores you've entered and displays it.

Reset Scores: Deletes all the scores you've entered and allows you to start the process all over again.
